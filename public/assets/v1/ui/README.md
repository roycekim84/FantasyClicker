# v1 PNG UI Skin

This folder contains individual PNG assets used by the v1 PNG skin.

Naming conventions:
- panels/panel_main.png: main panel frame (9-slice)
- panels/panel_sub.png: sub-panel frame (9-slice)
- buttons/btn_primary.png: primary button frame (9-slice)
- buttons/btn_primary_hover.png: primary hover frame (9-slice)
- buttons/btn_primary_active.png: primary active frame (9-slice)
- buttons/tab_idle.png: tab idle frame (9-slice)
- buttons/tab_active.png: tab active frame (9-slice)
- bars/bar_track.png: progress/track frame (9-slice)
- icons/placeholder.png: temporary icon placeholder

All PNGs are expected to be 9-slice friendly. Replace these placeholders with final art later.
